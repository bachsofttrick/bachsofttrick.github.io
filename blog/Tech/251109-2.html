watch with colors

Nov 9, 2025

After the previous long post, this one is shorter.

I was reading this post. And wanted to use gpustat as a way to continuously view GPU utilization. But using watch -n1 gpustat would wash away the colors. And using -c was not helping. Luckily, lol (yes, that's a handle) had a solution on Stack Overflow. It was to use unbuffer from expect package. So it's watch -n1 -c unbuffer gpustat and it will show colors.
