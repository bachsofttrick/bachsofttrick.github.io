Yesterday, I had a chance to use Gemma 4 through Cerebras. I used it through OpenCode. This is my first time trying out a different coding agent. Very fast inference, almost real time. I was able to use Gemma to help me build a simple React Native app for learning mobile development. However it's not reliable. A lot of time, I prompted OpenCode, there was a running process, and then it stopped with no response. It could be a hang from Cerebras receiving too many requests, or maybe OpenCode received the wrong reasoning tag. I don't know. But it was nice while it lasted.
